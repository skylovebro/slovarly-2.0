# Slovarly 2.0 🚀

**Slovarly 2.0** — это легковесное десктопное приложение на Python для эффективного изучения иностранных слов. Современный интерфейс в темных тонах и система геймификации помогают сохранять мотивацию в процессе обучения.

## ✨ Основные функции

* **Интерактивное тестирование:** Проверка знаний в режиме «Значение -> Слово».
* **Система Streak:** Счетчик правильных ответов подряд во время теста.
* **Ударный режим (Daily Streak):** Отслеживание количества дней непрерывного обучения.
* **Визуализация прогресса:** Автоматическое построение графиков успеваемости за последние 7 сессий.
* **Управление словарем:** Быстрое добавление новых слов через встроенный интерфейс.

## 🛠 Технологический стек

* **Интерфейс:** [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter) (современный Dark UI).
* **Анализ данных:** [Pandas](https://pandas.pydata.org/) (обработка истории тестов в формате CSV).
* **Графика:** [Matplotlib](https://matplotlib.org/) (визуализация прогресса).
* **Хранение:** Текстовые файлы (TXT) и табличные данные (CSV).
